== Description ==

This plugin helps you with showing a cookie notice without giving anything away about your site. The frontpage is custimizable and scrips are blocked from running untill the cookie notice is accepted.

=This plugin is=
* Google and Bing crawler friendly
* Easy to use and quick to set up
* Customisable in form and function

== Installation ==
1. Upload the unpacked folder to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Go to the settings menu and activate the cookie notice.
